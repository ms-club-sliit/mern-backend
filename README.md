## MERN Stack Backend

This is a simple backend application to get started with MERN stack development. This repository include some fundamental concepts like **Arrow functions**, and **Promises** inside the `javascript-basics` :open_file_folder:. The target :dart: is to implement a simple backend application with database connection and complete CRUD operations.
